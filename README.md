# file-encrypter
The FileEncrypter is a .NET Framework application developed in C#. It provides a graphical user interface for encrypting and decrypting files.

# Building the Project
The project can be built using Visual Studio or MSBuild. The output of the build will be placed in the bin\Debug or bin\Release directory, depending on the configuration.

# Running the Application
After building the project, you can run the application by executing FileEncrypter.exe in the output directory. The application provides a graphical user interface for selecting files to encrypt or decrypt, and progress bars to indicate the progress of the operation.
